DLL Loader(dinput8.dll Proxy)

DLL Loader which will load all DLLs inside a folder

How to use

Firstly download the DLL, it should be named as dinput8.dll, then drag and drop it into the same folder as your game, then create a folder called "injected"

All DLLs inside the injected folder should now load once you start the game, only add DLLs you trust, as they are executable files.
