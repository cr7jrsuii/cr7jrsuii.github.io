Lego City Undercover Freecam

DLL mod that lets you use a freecam in Lego City Undercover

Xbox Controller required(workarounds needed for other controllers)

How to use

You will need a way of loading a DLL file, DLL files are executables eg .exe but they cannot be ran just by double clicking.

Option 1, DLL loader. For this check out DLL Loader, You will need to drag the freecam DLL into the injected folder.

Option 2, Cheat Engine. For this you will need Cheat Engine, Open the LCU process and click on Memory View, once you see the memory viewer window go to Tools > Inject DLL, here select the Freecam DLL and click Yes.

Option 3, System Informer(Formerly known as Process Hacker), For this you will need System Informer, Right click on LCU process and click on Properties, Then go on Modules and click on Options, and press Load Module, then select the Freecam DLL.

Option 4, Dedicated DLL Injector, You can use any DLL Injector and it should work.

Controls

F1 to toggle

Left joystick to move around

Right joystick to look around

Right trigger to move up

Left trigger to move down

Up Arrow key to increase speed

Down Arrow key to decrease speed

Right Arrow key to increase sensitivity

Left Arrow key to increase sensitivity
